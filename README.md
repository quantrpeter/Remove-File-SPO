# How to deploy

## Download sppkg

[https://github.com/quantrpeter/Remove-File-SPO/releases](https://github.com/quantrpeter/Remove-File-SPO/releases)

then deploy your app catalog

## Build it yourself

use node 24

```
npm run build
```

then deploy sharepoint/solution/remove-file-spo.sppkg to your app catalog
